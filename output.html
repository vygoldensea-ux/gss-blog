n8n là gì?

n8n là nền tảng workflow automation mã nguồn mở, cho phép kết nối hàng trăm ứng dụng và tự động hóa quy trình mà không cần code phức tạp.

Chuẩn bị

Bạn cần Docker hoặc Node.js 18+, tài khoản Gmail và Slack.

Bước 1 — Cài đặt

docker run -it --rm -p 5678:5678 n8nio/n8n

Bước 2 — Tạo workflow

Thêm Schedule Trigger → Google Sheets → Slack node, kéo thả kết nối chúng lại.

Bước 3 — Kích hoạt

Bật toggle Active — workflow chạy tự động từ đây.
