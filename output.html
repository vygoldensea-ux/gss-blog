n8n là gì?

n8n là nền tảng workflow automation mã nguồn mở, cho phép kết nối hàng trăm ứng dụng và tự động hóa quy trình làm việc mà không cần viết code phức tạp. Khác với Zapier hay Make.com, n8n có thể self-host miễn phí trên server riêng.

Chuẩn bị trước khi bắt đầu

Để bắt đầu với n8n, bạn cần chuẩn bị:

  • Máy tính cài Docker hoặc Node.js 18+
  • Tài khoản các app muốn kết nối: Gmail, Slack, Google Sheets
  • Khoảng 30 phút thực hành
💡 Tip: n8n phù hợp nhất với team có ít nhất 1 người biết JavaScript cơ bản để tận dụng Function nodes.

Bước 1 — Cài đặt n8n bằng Docker

Mở terminal và chạy lệnh sau để khởi động n8n trong vài giây:

docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n

Sau 30 giây, mở trình duyệt vào http://localhost:5678 và tạo tài khoản admin đầu tiên.

Bước 2 — Tạo workflow đầu tiên

Click New Workflow và thêm 3 node theo thứ tự:

  1. Schedule Trigger — chạy tự động mỗi sáng 8h
  2. Google Sheets — đọc danh sách khách hàng mới
  3. Slack — gửi báo cáo vào channel #sales

Bước 3 — Kết nối credentials và kích hoạt

Click vào từng node, chọn Create New Credential và đăng nhập tài khoản tương ứng. Sau khi test thành công bằng nút Execute Workflow, bật toggle Active ở góc trên — workflow sẽ chạy tự động từ giờ.

Lỗi thường gặp

  • OAuth callback fail: Kiểm tra redirect URL trong Google/Slack console
  • Không trigger đúng giờ: Kiểm tra timezone trong Settings → System
  • Rate limit: Thêm node Wait 1-2 giây giữa các API call
